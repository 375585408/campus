关于增开万福广场至雁山大学城免费专线的通知

全校师生员工：

桂林市公交公司定于2015年3月25日增开万福广场至雁山大学城免费专线（1号线、2号线），以方便师生出行。

具体运行路线及发车时间如下：

万福专线车1号线：

万福广场 D 二塘 D 旅专（进校园） D 会仙路口 D 雁山镇 D 理工大学（进校园，往返停靠站点相同）

万福专线车2号线：

（1）从万福广场往广西师大方向行车线路及停靠站点：

万福广场 → 二塘 → 旅专校门口 → 会仙路口 → 雁山区镇政府 → 漓江学院 → 广西师大

（2）从广西师大往万福广场方向行车线路及停靠站点：

广西师大 → 会仙路口 → 旅专校门口 → 二塘 → 万福广场

发车

周一 周二 周三 周四 周五 周六 周日
